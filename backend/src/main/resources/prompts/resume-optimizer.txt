# 输出格式
请严格以JSON格式输出，格式如下：
{
  "overallScore": 75,
  "suggestions": [
    {
      "dimension": "summary",
      "original": "original text",
      "optimized": "optimized text",
      "reason": "reason",
      "severity": "high"
    }
  ],
  "optimizedContent": "full optimized resume"
}

# 注意事项
- overallScore 为0-100的整数
- suggestions 数组中每条建议必须包含所有字段
- dimension 只能是：summary, work, project, skills, education, structure
- severity 只能是：high, medium, low
- optimizedContent 是应用所有建议后的完整简历文本

# 硬约束
- 只输出一个 JSON 对象，不要其他说明文字
- 确保输出是合法的JSON格式
